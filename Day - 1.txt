DataBase:

API:

	Application Programming Interface(API) is an interface or communication protocol between client and server.

http request:

- Hyper Text Transfer Protocol
- Http works as request-response protocol between a client and server.
- A client(browser) sends an HTTP rrequest to the server.
- Sends and Http request in form of xml/json.


http Response:

	The client submits an HTTP request to the server and after internalizing the message, the server sends back a response. The response contains status information about the request.

xml:

- Extensibile Markup Language.
- XML is platform independent language.

json:

- JavaScript Object Notation
- JSon in platform independent language.

- API helps as a bridge between client and server. It supports only xml/json and it is platform independent.

API:

- Application Programming Interface.
- Client --- Server.
- XML/JSON
- Platform independent.
